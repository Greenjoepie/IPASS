\doxysubsection{Detailed Description}
\mbox{\hyperlink{class_m_a_x7219lib}{MAX7219lib}} class. 

This class is used to operate an up to 2x2 grid of LED matrices powered by the MAX7219 chip. It has the ability to set a single pixel or a rectangle of pixels on the display, or clear it entirely. While compatible with smaller grids, it is intended to be used with a 2x2 grid.

This function writes data to the MAX7219 without its own spi\+\_\+transaction. This means the data will not get read by the MAX7219 until the external spi\+\_\+transaction is destroyed.
