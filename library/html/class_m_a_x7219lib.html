Detailed Description

MAX7219lib class.

This class is used to operate an up to 2x2 grid of LED matrices powered by the MAX7219 chip. It has the ability to set a single pixel or a rectangle of pixels on the display, or clear it entirely. While compatible with smaller grids, it is intended to be used with a 2x2 grid.

void MAX7219lib::no_op ( int  repeat)

Send no_op command.

This function sends a no_op code to the MAX7219 a repeated amount of times. This is used for copying data (like setup data) over all matrices, as just sending a no_op code will shift the data over to the next matrix while keeping it on the current one.

Write function without sending data.

This function writes data to the MAX7219 without its own spi_transaction. This means the data will not get read by the MAX7219 until the external spi_transaction is destroyed. This is useful for when you want to write multiple pieces of data to a single matrix, otherwise each subsequent piece of data will instead get sent to the next matrix.
